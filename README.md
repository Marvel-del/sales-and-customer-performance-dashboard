# Sales and Customer Performance Dashboard

## Overview
This project showcases an interactive Power BI dashboard developed to analyze sales performance and customer behavior across multiple dimensions, including age group, gender, product category, city, and payment method.

The dashboard demonstrates how data analysis and visualization techniques can be applied to generate actionable insights for decision-making across various sectors, not limited to construction or engineering-related fields.

## Objectives
- Analyze overall sales performance and revenue trends
- Understand customer behavior across demographic groups
- Identify high-performing product categories
- Evaluate the impact of payment methods on revenue and pricing
- Present insights through clear and interactive visualizations

## Tools & Technologies
- Power BI– Data modeling, DAX measures, and interactive dashboards
- Python– Data cleaning and preparation, Querying concepts and data validation
- Data Analysis & Reporting

## Key Insights
- Revenue distribution varies significantly across different age groups
- Average selling prices differ by product category
- Gender-based revenue contribution is relatively balanced
- Payment methods influence price variability and sales patterns
- Customer location impacts purchasing behavior
